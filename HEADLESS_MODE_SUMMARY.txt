⚙️ TECHNICAL DETAILS:
═══════════════════════════════════════════════════════════════

How it works:

1. User checks/unchecks "Show browser window"
2. Frontend passes headless=true/false to backend
3. Backend uses: chromium.launch({ headless: isHeadless })
4. Browser runs accordingly

Benefits:
• Per-session control
• No config files needed
• Simple boolean flag
• Works on all platforms

═══════════════════════════════════════════════════════════════

📊 PERFORMANCE:
═══════════════════════════════════════════════════════════════

Example with 50 scans:

Window Visible + Screenshots (☑☑): 30-65 seconds
Headless + Screenshots (☐☑):       27-62 seconds  
Headless + No Screenshots (☐☐):    25-50 seconds

Headless mode saves 2-5 seconds by avoiding GUI rendering.
